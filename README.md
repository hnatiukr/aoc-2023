Source: https://adventofcode.com/2023

## Usage

> replace `01` with a date from `01` to `25`

### Installation

```console
make
```

### Prepare files for a new day

```console
make new day=01
```

### Run challenge

```console
make run day=01
```

### Run tests

```console
make test day=01
```
